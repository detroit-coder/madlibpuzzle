nThe Baseline folder contains all of the files and configurations needed to fast-track the creation of a dynamic webpage. To create a new Baseline webpage, copy the Baseline folder and rename. DO NOT modify or rename the original Baseline folder or any items therein.

Baseline includes the following files:

index.html: HTML site skeleton with the following CDNs: Bootstrap CSS, Bootstrap JS, Tether, JQuery, JQuery UI, AngularJS, AngularJS Route. Linked to style.css and script.js

style.css: blank CSS document

script.js: blank JavaScript file

reset.css: format reset file. NOT linked to index.html by default


Baseline includes the following folders:

html/: to hold additional html pages

js/: to hold additional JavaScript files

img/: to hold images
